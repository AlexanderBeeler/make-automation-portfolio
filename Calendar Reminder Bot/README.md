# Calendar Reminder Bot

## Problem

People often forget to check their calendar throughout the day and want an easier way to stay informed about upcoming events.

## Solution

This Make automation searches Google Calendar for events occurring on the current day and sends a formatted schedule to Discord, including the event date, start time, end time, and a friendly "Good luck today!" message.

## Services Used

- Make
- Google Calendar
- Discord

## Skills Demonstrated

- Google Calendar integration
- Scheduled automation
- Date filtering
- Date formatting
- Dynamic data mapping
- Discord notifications

## Workflow

Google Calendar → Make → Discord
