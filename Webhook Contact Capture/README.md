# Webhook Contact Capture

## Problem

Businesses need an automated way to collect and organize website contact form submissions.

## Solution

This Make automation receives website form submissions through a webhook, stores the submitted information in Google Sheets, and sends a Discord notification whenever a new contact is received.

## Services Used

- Make
- Webhooks
- Google Sheets
- Discord

## Skills Demonstrated

- Webhook integration
- Data mapping
- Google Sheets integration
- Discord notifications
- Form automation

## Workflow

Webhook → Make → Google Sheets → Discord
