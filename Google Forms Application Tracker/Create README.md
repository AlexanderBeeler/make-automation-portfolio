# Google Forms Application Tracker

## Problem

Businesses need to know immediately when a new application has been submitted instead of constantly checking Google Forms.

## Solution

This Make automation watches Google Forms for new submissions and automatically sends a Discord notification containing the applicant's submitted information.

## Services Used

- Make
- Google Forms
- Discord

## Skills Demonstrated

- Google Forms integration
- Trigger-based workflows
- Dynamic field mapping
- Discord notifications

## Workflow

Google Forms → Make → Discord
