# Google Drive File Logger

## Problem

Businesses that manage large numbers of files often need an easy way to track newly uploaded documents without manually checking Google Drive.

## Solution

This Make automation monitors a selected Google Drive folder for newly uploaded files. When a new file is detected, it logs the file name, owner, creation date, and web view link into Google Sheets for quick access and record keeping.

## Services Used

- Make
- Google Drive
- Google Sheets

## Skills Demonstrated

- Google Drive integration
- File monitoring
- Metadata extraction
- Dynamic data mapping
- Google Sheets integration

## Workflow

Google Drive → Make → Google Sheets
