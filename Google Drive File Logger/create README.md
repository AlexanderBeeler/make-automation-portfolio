Google Drive File Logger
Problem

Businesses that manage large numbers of files often need an easy way to track newly uploaded documents without manually checking Google Drive.

Solution

Built a Make automation that monitors a selected Google Drive folder for new files, logs the file name, owner, creation date, and web view link into Google Sheets for quick access and record keeping.
